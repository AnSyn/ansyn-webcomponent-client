Import Ansyn Component Into Angular App

  1. copy the folder content into new folder inside your root project folder.
  2. inside app.module.ts add CUSTOM_ELEMENT_SCHEMA to schemas array.
  3. inside angular.json add to your project assets array: 
    		{{'{'}}
			"input": "<name_of_folder_from_step1>/assets",
			"output": "/assets"
			{{'}'}},
			{{'{'}}
			"input": "<name_of_folder_from_step1>/res",
			"output": "/res"
			{{'}'}}
  4. inside angular.json add to your project scripts array: "<name_of_folder_from_step1>/custom-ansyn.min.js"
  5. you can use <custom-ansyn></custom-ansyn> inside your app.
